# DFM 

どOふろ制御用ライブラリ

# Quick start

仮想環境を作る&有効化

```bash
python3 -m venv venv
source venv/bin/activate
```

AEMをインストール

```bash
https://github.com/Sachi854/AndroidEmuMacro.git
cd AndroidEmuMacro
python ./setup.py install
pip install -r requirements.txt
```

適当にdfm以下をimport

```python
import dfm.base as bs
```

あとは頑張れ
